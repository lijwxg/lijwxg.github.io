亚马逊云是十分领先的云服务厂商, 提供了大量开箱即用服务.大大减少了开发成本.

使用亚马逊的云服务

EC2

Elastic Compute Cloud 服务器

S3

Simple Storage Service 存储

ECR

Elastic Container Registry 容器注册表

Lambda

Serverless, 函数

SageMaker

大规模构建、训练和部署机器学习模型

aws工具集

安装awscli

1
2
3

mkvirtualenv aws
workon aws
pip install awscli

配置系统环境

从我的安全凭证中获取到access key id, secret access key

1

aws configure

授权ec2访问s3

在我的安全凭证中创建角色, 给EC2服务增加S3的权限

1
2

aws s3 ls
aws s3 cp s3://lijwxg/demo .

使用ecr

登陆

1

aws ecr get-login --no-include-email|sh

推送本地镜像到ecr

1
2
3

docker images
docker tag 338f12981185 111111111.dkr.ecr.ap-northeast-1.amazonaws.com/demo
docker push 111111111.dkr.ecr.ap-northeast-1.amazonaws.com/demo
